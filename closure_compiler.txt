Скомпилировать нессколько js-файлов в один:

ЗАПУСК ИЗ КОМАНДНОЙ СТРОКИ 
1) открыть командную строку cmd
2) зайти в папочку с файлом compiler.jar
	cd Progmar Files/nodejs/node_modules/
3) проверить корректность установки и запуска компилятора можно командой:
	java -jar compiler.jar --help
	
4) если всё корректно, продолжаем работу: 
	java -jar compiler.jar --js file1.js --js file2.js --js file3.js --js_output_file test.js --compilation_level SIMPLE_OPTIMIZATIONS (ADVANCED_OPTIMIZATIONS|WHITESPACE_ONLY)
	
	где file1.js, file2.js, file3.js - файлы для сборки
	test.js - файл конечного результата
	

Насчет compilation_level: для нас совсем не подходит уровень ADVANCED_OPTIMIZATIONS - он не предполагает наличие глобальных переменных и будет вырезать весь код, который не вызывается в явном виде.
В режиме SIMPLE_OPTIMIZATION будут достигаться следующие цели:
- переименование локальных переменных
- объединение констант (если есть выражение типа 60*60, то компилятор сразу посчитает результат)
- удаление и оптимизация лишних переменных
- удаление лишних присваиваний
- объединение объявлений переменных
- преобразование простых условных выражений к тернарным операторам
- удаление кода после return
- и тд

на мой взгляд, этот уровень оптимизации наиболее подходит для проекта.

ЗАПУСК ИЗ IDE
1) открыть встроенную консоль (у меня Ctrl+Shift+X)
2) убедиться в том, то файл compiler.jar лежит в каталоге, из которого открыта консоль
3) выполнить аналогичные команды
